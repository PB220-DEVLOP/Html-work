What is audio tag

The HTML tag is used to embed sound content, like music or sound effects, directly into a web page. It allows users to play, pause, and control the volume of the audio files within the webpage The src attribute is the path to the audio you want to import. The controls attribute enables the audio player control features such as the play button , pause button , volume control bar , fast forward button , and the back forward button buttons.

what is iframe tag

The HTML tag, which stands for "inline frame," is used to embed another HTML document within the current HTML document. This creates a nested browsing context, essentially displaying a separate web page or content from another source within a designated area of the main page.

Video tag

The video tag in HTML5 is used to embed video content directly into a web page without needing external plugins like Flash. It allows users to view videos with built-in browser controls such as play, pause, volume, and fullscreen options. The tag is simple to use and supports various video formats including MP4 (video/mp4), WebM (video/webm), and Ogg (video/ogg), ensuring compatibility across different browsers. The basic structure includes the video tag with optional attributes and one or more source tags inside it, specifying the file location and format. Common attributes include controls (to display playback controls), autoplay (to play the video automatically), loop (to replay the video after it ends), muted (to start the video without sound), and poster (to display a thumbnail before the video starts). Width and height can also be specified.

The HTML embed tag is used to embed external content or interactive media directly into a web page. This content can include various formats such as images, audio files, video files, PDF documents, or even applications that require a browser plugin (though plugin usage is increasingly limited due to security concerns).
